Relative
Absolute positioned divs look for a parent element with a defined position property to position against. if it can't find one, positions absolutely to the body tag
Fixed behaves similarly to absolute but will be fixed on the screen regardless

when we have a relative parent, absolute or fixed position child elements will position in relation to the relatively positioned parent
